## Running Locally
Clone this repository and install dependencies:

```bash
git clone git@github.com:Mudinyika/mudiwa-portfolio.git
cd mudiwa-portfolio
npm install
npm run dev
```


Open [http://localhost:4321] in your browser to view locally.
